Ayuda: Cómo usar la aplicación

Esta aplicación te permite crear, editar y gestionar un checklist interactivo en formato Markdown. Aquí tienes un recorrido de usuario para entender cómo funciona:

  1. Crear un checklist desde cero: Haz clic en el botón ✏️ Editar checklist para abrir el editor de Markdown. Escribe tu checklist en formato Markdown y haz clic en Actualizar Checklist para renderizarlo.
  2. Interactuar con el checklist: Marca o desmarca las casillas directamente en el área interactiva. El progreso se actualizará automáticamente.
  3. Cargar un archivo Markdown: Haz clic en 📂 Cargar .md para cargar un archivo Markdown existente.
  4. Guardar el checklist: Haz clic en 💾 Descargar .md para guardar el checklist como un archivo Markdown.
  5. Descargar el html renderizado: Haz clic en 💾 Descargar Renderizado para guardar el checklist como un archivo html dinámico reutilizable.
